# Logo Assets and Color Documentation

## Cats Effect Logo

### Official Logo URL
- SVG: https://typelevel.org/cats-effect/img/cats-effect-logo.svg
- Repository: https://github.com/typelevel/cats-effect/tree/main/images

### Color Palette (Extracted from SVG)

Based on analysis of the official SVG logo, Cats Effect uses a soft, pastel color palette:

1. **Teal/Cyan (Background Hexagon)**: `#6dd3ce`
   - RGB: rgb(109, 211, 206)
   - Usage: Primary background shape, hexagonal frame

2. **Slate Blue (Cat Body)**: `#53729a`
   - RGB: rgb(83, 114, 154)
   - Usage: Cat illustration, main body

3. **Coral/Peach (Books/Pages)**: `#f7a278`
   - RGB: rgb(247, 162, 120)
   - Usage: Books, decorative elements

4. **Soft Pink (Accents)**: `#f1beca`
   - RGB: rgb(241, 190, 202)
   - Usage: Secondary accents, highlights

5. **Black (Strokes/Outlines)**: `#000000`
   - Usage: Line work, borders, stroke definitions

### Logo Description
The Cats Effect logo features:
- A hexagonal frame in teal (#6dd3ce)
- A stylized cat illustration in slate blue
- Stack of books representing the "Effect" library
- Clean, modern illustration style

### Usage Guidelines
- Available in SVG format for scalability
- Transparent background suitable for light/dark themes
- Official logo hosted at typelevel.org/cats-effect

## ZIO Logo

### Official Logo URLs
- PNG (Website): https://zio.dev/img/zio.png
- Repository Assets: https://github.com/zio/zio/tree/master/website/static/img
- Full Brand Assets: https://github.com/zio/zio/tree/master/website/static/img/zio-RGB

### Color Palette (Extracted from Official Logo)

Based on visual analysis of the official ZIO logo:

1. **Bright Red (Primary)**: `#FF3300` to `#FF6600`
   - RGB: rgb(255, 51, 0) to rgb(255, 102, 0)
   - Usage: ~55% of logo, main "Z" and "I" strokes
   - Note: Gradient effect with oranges

2. **Dark Maroon**: `#990000`
   - RGB: rgb(153, 0, 0)
   - Usage: ~20% of logo, darker contrasting elements

3. **Orange (Secondary)**: `#FF6600`
   - RGB: rgb(255, 102, 0)
   - Usage: ~20% of logo, gradient transitions

4. **Yellow (Accent)**: `#FFFF00`
   - RGB: rgb(255, 255, 0)
   - Usage: ~5% of logo, tip accents

5. **Common Brand Orange**: `#e73c00` (Observed in website elements)
   - RGB: rgb(231, 60, 0)
   - Usage: Website CTAs, buttons, brand elements

### Logo Description
The ZIO logo features:
- Bold, stylized "ZIO" text
- Dynamic gradient from red → orange → yellow
- Energetic, modern aesthetic
- Strong visual impact with warm color palette

### Available Formats
ZIO provides comprehensive brand assets in:
- **PNG**: Various sizes (navbar_brand.png, navbar_brand2x.png)
- **SVG**: Vector format for scalability
- **RGB**: Full color and monochrome variants
- **CMYK**: Print-ready versions (zio-full-color-cmyk)
- **EPS/PDF/AI**: Professional design formats
- **Specialized**: GitHub, Twitter, splash versions
